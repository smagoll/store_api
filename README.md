# 🛍️ Store API

**Store API** — учебный проект интернет-магазина на **ASP.NET Core (C#)**.  
Реализует базовый функционал онлайн-магазина: регистрация и авторизация пользователей, просмотр товаров, корзина и оформление заказов.  

Проект построен по принципам многослойной архитектуры и ориентирован на использование в качестве backend-API для веб- и мобильных приложений.  

---

## ✨ Возможности
- 🔐 Регистрация и авторизация (JWT + refresh-токены)  
- 👤 Роли пользователей (**Admin**, **User**)  
- 📦 Управление товарами: список, детали  
- 🛒 Корзина: добавление, удаление, изменение количества  
- 💳 Оформление заказов  
- ⚙️ REST API для интеграции с frontend или мобильными приложениями  
- 📝 Swagger UI для удобного тестирования  

---

## 🏗️ Архитектура
Проект реализован по многослойной архитектуре:  

- **Controllers** — обработка HTTP-запросов  
- **Services** — бизнес-логика  
- **Repositories** — доступ к данным (EF Core)  
- **DTOs / Models** — объекты передачи данных и сущности БД  
- **AutoMapper** — маппинг DTO ↔ Entities  

---

## 🗄️ База данных
- **Entity Framework Core** — работа с БД  
- **PostgreSQL** — основная СУБД  
- Поддерживаются **миграции** и **начальные данные (seed)**  

📌 В проекте есть схема БД (ER-диаграмма).  

---

## 🔐 Аутентификация и авторизация
- JWT-токены для авторизации  
- Refresh-токены для продления сессии  
- Ролевая модель доступа (**Admin**, **User**)  

---

## 🧪 Тестирование
- Unit-тесты на **xUnit**  
- Используется **Moq** для имитации зависимостей  

---

## 📊 Логирование
- **Serilog** — структурированное логирование  

---

## ✅ Валидация
- **FluentValidation** — валидация DTO перед обработкой  

---

## 🛠️ Технологии
- C# / ASP.NET Core  
- Entity Framework Core  
- PostgreSQL  
- JWT (аутентификация)  
- AutoMapper  
- FluentValidation  
- Serilog
